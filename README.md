# Website UI Overrides

Rules that I have created while using various websites to improve usability. 
Frustrated with the UI of various sites, I decided to take matters into my 
own hands by writing custom CSS and JavaScript overrides.

## How to Use

To enable these custom overrides, use the 
[User JavaScript and CSS](https://tenrabbits.github.io/user-js-css-docs/) 
browser extension for Chrome or Edge. Other similar extensions may also work.

Each subfolder contains site-specific rules. Review the included `README.md` 
files or comments in each directory for details on what each override does.
